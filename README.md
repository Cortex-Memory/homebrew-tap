# Cortex Memory Homebrew Tap

Official Homebrew tap for [Cortex Memory](https://cortexmemory.dev) CLI tools.

## Installation

```bash
brew install cortex-memory/tap/cli
```

Or tap first, then install:

```bash
brew tap cortex-memory/tap
brew install cli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `cli` | CLI for managing Cortex Memory deployments |

## Updating

```bash
brew update
brew upgrade cortex-memory/tap/cli
```
